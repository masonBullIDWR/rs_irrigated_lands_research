# rs_irrigated_lands_research
Collection of codes for data analysis and visualization of various rs methods. Originally created for the Treasure Valley Irrigated Lands Study, 2026.

# This is the code that did the Treasure Valley Comparison Study
The first thing that needs to run is *get_area_over_time_for_multiple_datasets_to_csv.py*. This only needs to be run once. 
Next run *add_hd_and_rf_to_area_csv_and_plot.py* to get a plot of area over time for urban and irrigated area. You can choose in this script whether or not to include bimodal change classification areas as well.

These scripts do not create any rasters, these codes are predicated on you already having classified rasters. I will add the rasters that I created for the study in an update.
